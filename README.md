# Setinfo-Stealer-Rework


Плагин для AMX Mod X, который собирает информацию об игроках на сервере Counter-Strike 1.6 и логирует её в удобном виде. Записывает setinfo/userinfo данные (в том числе пароли от админок, VIP и других плагинов), сообщения из чата, а также события входа/выхода игроков.

---

## Возможности

- 📋 Логирование setinfo и userinfo данных каждого игрока при заходе на сервер
- 💬 Логирование всех сообщений из чата (общий и командный)
- 🚪 Логирование событий подключения и отключения игроков
- 🔑 Подсветка найденных паролей в консоли сервера
- 📁 Раздельные логи для каждого игрока по SteamID
- 📜 Общий лог со всеми игроками для удобного просмотра
- ⚙️ Пропуск дефолтного флага `z`, чтобы не мусорить в логах

---

Содержит данные ВСЕХ игроков, заходивших на сервер
Содержит все сообщения игрока + события входа/выхода
Если у игрока найден пароль — выводится одна короткая строка в консоль сервера
Если паролей нет — в консоль ничего не выводится, чтобы не спамить

## Возможные проблемы

### Папки не создаются автоматически

На некоторых сборках AMXX функция `mkdir` может не работать. Если папок нет:

1. Создай их вручную:
amxmodx/logs/setinfo_stealer/
amxmodx/logs/setinfo_stealer_chat/

2. Перезапусти сервер.

### В консоли пишет `Cannot open file: ...`

Это значит, что у плагина нет прав на запись в папку логов. Проверь права доступа к папке `amxmodx/logs/`
